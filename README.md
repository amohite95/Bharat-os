## 🚀 The Micro-Multinational Architecture
This repository contains the core operational infrastructure for `am_labs`, built entirely from a mobile node with zero-cost architecture.

### ⚙️ Core Engines Online:
1. **`core_engine.py`**: The autonomous heartbeat validating OS uptime.
2. **`api_gateway.py`**: Local HTTP server hosting am_labs company data on Port 8080.
3. **Cloudflare Tunneling**: Secure global bridging to expose the local mobile node to the internet (Edge location: bom03 - Mumbai).
